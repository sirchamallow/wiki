# Logiciels

Récemment j'ai découvert qu'il existait **un gestionnaire de paquet pour Windows**. Son nom ? [**Chocolatey** ](https://chocolatey.org/)! 

Il vous sera ainsi possible d'installer en un temps records, Mozilla Firefox, Google Chrome, CCleaner et bien d'autres! Et pour les allergiques à la ligne de code il existe [**Chocolatey-GUI**](https://chocolatey.org/packages/ChocolateyGUI), qui est une interface graphique pour le gestionnaire :\) . 

Pour comprendre un peu mieux le fonctionnement je vous invite à lecture des billets de différents blogueurs : [Le Crabe](https://lecrabeinfo.net/chocolatey-gestionnaire-paquets-windows-installer-logiciels-applications-ligne-commande.html) et [Tech2Tech](https://www.tech2tech.fr/chocolatey-votre-gestionnaire-de-paquets-sous-windows/).

## Logiciels Windows

### Installation via Chocolatey

Trêve de clavardage, voici une liste de logiciels que je vous recommande 

| **Nom** | **Remarque** | **Raccourci chocolatey** |
| :--- | :--- | :--- |
| 7zip | Gestionnaire d'archives opensource | &gt; choco install 7zip |
| AdwCleaner \(portable\) | pour supprimer les adwares, bloatwares, barres d'outils et autres programmes indésirables | &gt; choco install adwcleaner |
| Bitwarden | Gestionnaire de mot de passe opensource | &gt; choco install bitwarden |
| Brave Browser | Navigateur opensource orientée vie privée | &gt; choco install brave |
| CCleaner | Nettoyer Windows10 \(avec précaution\) | &gt; choco install ccleaner |
| Chromium | Version opensource du navigateur Chrome | &gt; choco install chromium |
| CPU-Z | Utilitaire système permet d'afficher certaines caractéristiques de son matériel | &gt; choco install cpu-z |
| Defraggler | Défragmenter une disque dur \(qui n'est pas un SSD\) | &gt; choco install defraggler |
| Disable Nvidia Telemetry | Désactive la télémétrie des drivers nvidia | &gt; choco install disable-nvidia-telemetry |
| f.lux | Optimise l'affichage de votre écran en fonction de l'heure de la journée | &gt; choco install f.lux |
| Foxit Reader | Logiciel opensource de lecture de fichiers PDF | &gt; choco install foxitreader |
| Git | Ben git quoi... :\) | &gt; choco install git |
| GOG Galaxy | Plateforme GOG \(jeux vidéo\) | &gt; choco install goggalaxy |
| Greenshot | LE meilleur logiciel pour vos captures d'écrans | &gt; choco install greenshot |
| Molotov.tv | Regarder la télévision française en direct \(gratuit et sécurisée\) | &gt; choco install molotov |
| Mozilla Firefox | Navigateur opensource orientée vie privée | &gt; choco install firefox |
| Mozilla Thunderbird | Logiciels de messagerie opensource | &gt; choco install thunderbird |
| Net Balancer | pour surveiller votre bande passante | &gt; choco install netbalancer |
| Origin | Plateforme EA \(jeux vidéo\) | &gt; choco install origin |
| Playnite | Bibliothèque opensource pour retrouver vos jeux sur les différentes plateformes | &gt; choco install playnite |
| Rufus | Pour créer clé bootable Linux | &gt; choco install rufus |
| Speccy | Utilitaire système permet d'afficher certaines caractéristiques de son matériel | &gt; choco install speccy |
| Steam | Plateforme de Valve \(jeux vidéo\) | &gt; choco install steam |
| Uplay | Plateforme Ubisoft \(jeux vidéo\) | &gt; choco install uplay |
| VLC Media Player | Lecteur vidéo opensource | &gt; choco install vlc |
| Visual Studio Code | IDE opensource de Microsoft | &gt; choco install vscode |
| Windirstat | Analyser la consommation d'espace-disque du PC | &gt; choco install windirstat |
