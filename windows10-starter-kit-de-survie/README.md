# Windows10 starter-kit

Que ce soit après une installation de [**Windows 10**](https://www.microsoft.com/fr-fr/store/b/windows) sur votre ordinateur, ou en le lançant la première fois suite à l'achat d'une tour fixe ou pc portable; le résultat est le même : votre prochaine étape quasi-obligatoire consistera à remettre "clean" votre système d'exploitation et désinstaller les applications inutile que Microsoft pré-installe par défaut sans votre accord.   
  
Sans oublier dans certains cas, celles installer par le fabricant de votre ordinateur.

## Checklist

Voici un checklist des bonnes pratiques à effectuer \(point de vue personnel\). Chaque point fera l'objet prochainement d'une page dédié.

* Remplacer ses DNS
* Remplacer le serveur NTP
* Désactiver les logiciels pré-installer par Windows10
* Mettre en place un firewall
* Faire les différentes mise à jour
* Personnaliser l'interface de windows
* Identification sur vos logiciels \(login, mot de passe etc.\)
