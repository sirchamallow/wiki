# DoH dans Mozilla Firefox

## Description

Voici comment activer la prise en charge du DNS sur HTTPS \(DoH - DNS-over-HTTPS\) dans le navigateur Firefox. Ce navigateur est actuellement le seul a supporter Le protocole DoH\*, qui permet de chiffrer en profondeur vos communications.

\(\*\) cette fonction n'est pas activée par défaut pour les utilisateurs de Firefox/
