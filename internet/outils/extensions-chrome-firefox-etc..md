# Extensions Chrome, Firefox etc.

Voici quelques extensions pour rendre plus agréable votre session de surf sur vos navigateurs internet de bureau. 

{% hint style="info" %}
Notes : si vous utilisez le navigateur **Brave** ou **Vivaldi**, il faudra utiliser les extensions de Chrome \(elle sont compatibles car repose sur le même moteur\)
{% endhint %}

Bloquer les publicités : **uBlock Origin**  
-&gt; [Chrome](https://chrome.google.com/webstore/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm) / [Firefox](https://addons.mozilla.org/fr/firefox/addon/ublock-origin/) / [Safari](https://github.com/el1t/uBlock-Safari#installation) / [Edge](https://www.microsoft.com/store/p/app/9nblggh444l4)

Surfer en HTTPS le plus souvent possible : **HTTPSeverywhere**  
-&gt; [Chrome](https://chrome.google.com/webstore/detail/gcbommkclmclpchllfjekcdonpmejbdp) / [Firefox](https://www.eff.org/files/https-everywhere-latest.xpi) / [Opera](https://addons.opera.com/extensions/details/https-everywhere/)

Se protéger des malwares, scams, clickbait : **Malwarebytes Browser Extension**  
-&gt; [Chrome](https://chrome.google.com/webstore/detail/malwarebytes-browser-exte/ihcjicgdanjaechkgeegckofjjedodee) / [Firefox](https://addons.mozilla.org/fr/firefox/addon/malwarebytes/)

Pare-feu matriciel : **uMatrix**  
-&gt; [Chrome](https://chrome.google.com/webstore/detail/umatrix/ogfcmafjalglgifnmanfmnieipoejdcf) / [Firefox](https://addons.mozilla.org/fr/firefox/addon/umatrix/) / [Opera](https://addons.opera.com/en-gb/extensions/details/umatrix/)

Gérer ses mots de passe : **Bitwarden \(open-source\)**  
-&gt; [Chrome](https://chrome.google.com/webstore/detail/bitwarden-free-password-m/nngceckbapebfimnlniiiahkandclblb) / [Firefox](https://addons.mozilla.org/firefox/addon/bitwarden-password-manager/) / [Safari](https://safari-extensions.apple.com/details/?id=com.bitwarden.safari-LTZ2PFU5D6) / [Edge](https://www.microsoft.com/store/apps/9P6KXL0SVNNL) / [Opera](https://addons.opera.com/extensions/details/bitwarden-free-password-manager/)

Empêcher certains sites de faire miner de la cryptomonnaie via votre CPU : **NoCoin**  
-&gt; [Chrome](https://chrome.google.com/webstore/detail/no-coin/gojamcfopckidlocpkbelmpjcgmbgjcl) / [Firefox](https://addons.mozilla.org/fr/firefox/addon/no-coin/) / [Opera](https://addons.opera.com/en-gb/extensions/details/no-coin/)

Remplir le captcha automatiquement : **Buster: Captcha Solver for Humans**  
-&gt; [Chrome](https://chrome.google.com/webstore/detail/buster-captcha-solver-for/mpbjkejclgfgadiemmefgebjfooflfhl) / [Firefox](https://addons.mozilla.org/fr/firefox/addon/buster-captcha-solver/) / [Opera](https://addons.opera.com/en/extensions/details/buster-captcha-solver-for-humans/)

Marre des pop-ups concernant les cookies : **I don’t care about cookies**  
-&gt; [Chrome](https://chrome.google.com/webstore/detail/i-dont-care-about-cookies/fihnjjcciajhdojfnbdddfaoknhalnja) / [Firefox](https://addons.mozilla.org/fr/firefox/addon/i-dont-care-about-cookies/) / [Opera](https://addons.opera.com/en/extensions/details/i-dont-care-about-cookies/) / [Pale Moon](https://addons.palemoon.org/addon/i-dont-care-about-cookies/)

Les vidéo en autoplay vous rendent dingue ? **Autoplay No More**  
-&gt; [Chrome](https://chrome.google.com/webstore/detail/autoplay-no-more/obiogedpmdnfaldjdjmnbpmhjjfnghnj) / [Firefox](https://addons.mozilla.org/fr/firefox/addon/autoplay-no-more/)

Trouver en un clic la version cache d’une page web sur les moteurs de recherche : **View Page Archive & Cache**  
-&gt; [Chrome](https://chrome.google.com/webstore/detail/view-page-archive-cache/hkligngkgcpcolhcnkgccglchdafcnao) / [Firefox](https://addons.mozilla.org/en-US/firefox/addon/view-page-archive/) / [Opera](https://addons.opera.com/en/extensions/details/view-page-archive-cache/)

Créer, modifier et changer rapidement de groupe d’onglets : **Simple Tab Groups**  
-&gt; [Firefox](https://addons.mozilla.org/fr/firefox/addon/simple-tab-groups/)

Analyser un fichier avec VirusTotal avant de télécharger : **VTchromizer / VTzilla**  
-&gt; [Chrome](https://chrome.google.com/webstore/detail/vtchromizer/efbjojhplkelaegfbieplglfidafgoka) / [Firefox](https://addons.mozilla.org/en-US/firefox/addon/vtzilla/)
