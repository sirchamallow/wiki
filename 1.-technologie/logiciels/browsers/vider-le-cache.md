# Vider le cache

Voici les étapes pour vider le cache sur les versions desktop et mobile des navigateurs Firefox, Chrome, Safari et Microsoft Edge

### Google Chrome

1. Ouvrez Chrome
2. Cliquez sur les **3 points (⋮)** en haut à droite
3. Cliquez sur **Paramètres**
4. Allez dans **Confidentialité et sécurité**
5. Cliquez sur **Effacer les données de navigation**
6. Cochez :
   * ✅ _Images et fichiers en cache_
   * (optionnel) _Cookies et autres données de site_
7. Sélectionnez la période (_Toutes les périodes_ recommandé)
8. Cliquez sur **Effacer les données**

👉 Raccourci clavier : `Ctrl + Shift + Suppr`

***

### Mozilla Firefox

1. Ouvrez Firefox
2. Cliquez sur le **menu ☰** en haut à droite
3. Cliquez sur **Paramètres**
4. Accédez à **Vie privée et sécurité**
5. Dans **Cookies et données de sites**, cliquez sur **Effacer les données**
6. Cochez :
   * ✅ _Contenu web en cache_
7. Cliquez sur **Effacer**

👉 Raccourci clavier : `Ctrl + Shift + Suppr`

***

### Safari (macOS uniquement)

#### Méthode rapide (cache uniquement)

1. Ouvrez Safari
2. Cliquez sur **Safari > Réglages (Préférences)**
3. Onglet **Avancées**
4. Activez **Afficher le menu Développement**
5. Dans la barre de menu :
   * Cliquez sur **Développement**
   * Puis **Vider les caches**

#### Méthode alternative (cache + cookies)

1. Menu **Historique**
2. Cliquez sur **Effacer l’historique**
3. Sélectionnez _toute l’historique_

***

### Vivaldi

1. Ouvrez Vivaldi
2. Cliquez sur le logo **Vivaldi (V)** en haut à gauche
3. Cliquez sur **Outils > Supprimer les données de navigation**
4. Cochez :
   * ✅ _Cache_
5. Choisissez la période (_Toutes les données_)
6. Cliquez sur **Supprimer**

👉 Raccourci clavier : `Ctrl + Shift + Suppr`

***

### Brave

1. Ouvrez Brave
2. Cliquez sur le **menu (≡)** en haut à droite
3. Cliquez sur **Paramètres**
4. Allez dans **Confidentialité et sécurité**
5. Cliquez sur **Effacer les données de navigation**
6. Cochez :
   * ✅ _Images et fichiers en cache_
7. Choisissez la période (_Toutes les périodes_)
8. Cliquez sur **Effacer les données**

👉 Raccourci clavier : `Ctrl + Shift + Suppr`

***

### Microsoft Edge

1. Ouvrez Edge
2. Cliquez sur les **3 points (⋯)** en haut à droite
3. Cliquez sur **Paramètres**
4. Accédez à **Confidentialité, recherche et services**
5. Sous **Effacer les données de navigation**, cliquez sur **Choisir les éléments à effacer**
6. Cochez :
   * ✅ _Images et fichiers en cache_
   * (optionnel) _Cookies et autres données de site_
7. Sélectionnez la période (_Toutes les périodes_)
8. Cliquez sur **Effacer maintenant**

👉 Raccourci clavier : `Ctrl + Shift + Suppr`

***

## Bonnes pratiques

* 🔄 **Redémarrer le navigateur** après nettoyage
* 🌐 **Forcer le rechargement d’une page** :\
  `Ctrl + F5`
* ⚠️ Supprimer les cookies peut entraîner une **déconnexion des comptes**
