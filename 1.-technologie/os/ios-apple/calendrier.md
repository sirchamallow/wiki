# Calendrier

## Définir Google Agenda comme calendrier par défaut

{% hint style="info" %}
Au préalable : Il faut d’abord ajouter votre compte Google à l’iPhone : Réglages > Calendrier > Comptes > Ajouter un compte > Google, puis activez « Calendrier » lors de la configuration.
{% endhint %}

Pour définir **Google Agenda comme calendrier par défaut sur iOS**, suivez ces étapes :

1. Ouvrez **Réglages** (ou « Paramètres ») sur votre iPhone.
2. Faites défiler jusqu’à **Calendrier**.
3. Touchez **Calendrier par défaut**.
4. Sélectionnez votre **calendrier Google** dans la liste proposée.

Dès lors, tous les nouveaux événements créés via l’application Calendrier de l’iPhone seront ajoutés sur votre Google Agenda par défaut :) .

{% hint style="success" %}
Cette manipulation ne remplace pas l’application Calendrier par celle de Google, mais tous les événements créés sur l’iPhone le seront dans Google Agenda.&#x20;
{% endhint %}

{% hint style="warning" %}
&#x20;iOS ne permet pas de définir l’application « Google Agenda » comme application par défaut pour ouvrir tous les liens **.ics** ou gérer tous les rappels (ce réglage concerne l’application Calendrier d’Apple uniquement).
{% endhint %}
